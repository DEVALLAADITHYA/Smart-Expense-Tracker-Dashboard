## Overview

The **Smart Expense Tracker Dashboard** is an interactive web application built using **Python and Streamlit** that helps users track and visualize their daily expenses. The dashboard provides insights into spending patterns through interactive charts and summaries, enabling better financial awareness and budgeting.

The application reads expense data from a CSV file and presents it through dynamic visualizations using Plotly.

---

## Features

* 📊 Interactive dashboard for expense analysis
* 📈 Visual representation of expenses using charts
* 📅 Category-wise and date-wise expense tracking
* 🧾 Simple CSV-based data storage
* ⚡ Fast and lightweight Streamlit interface

## Installation

### 1. Clone the repository

```
git clone https://github.com/DEVALLAADITHYA/Smart-Expense-Tracker-Dashboard.git
cd Smart-Expense-Tracker-Dashboard
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
streamlit run app.py
```

---

## Usage

1. Launch the application using Streamlit.
2. The dashboard loads expense data from the CSV file.
3. Users can view spending patterns through charts and analytics.

---

## Deployment

This application can be deployed easily using **Streamlit Cloud**:

1. Push the project to GitHub
2. Go to https://share.streamlit.io
3. Connect your repository
4. Select **app.py** as the main file
5. Deploy the app
